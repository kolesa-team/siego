Siego
=====

Siego is regression test and benchmark utility. Golang port of [Siege](https://github.com/JoeDog/siege) with extended stats output.

## Requirements

 * Go 1.6+
 * make
 * build-essential
 * fpm (`apt-get install ruby-dev && gem install fpm`) to generate *.deb

## Installation

    # make dep-install
	# make
	# make install
